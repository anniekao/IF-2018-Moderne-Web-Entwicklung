About This Website

This site was created as part of the course "Einführung in die moderne Web-Entwicklung/Intro to modern Web Development" held on August 6th - 8th at the 21st Informatica Feminale, Universität Bremen.

In order to obtain a credit point, participants were required to complete one of three assignments:

  1. HTML: create a new subpage in the existing project and rebuild the table found on the following website in HTML5:
    http://www.tushar-mehta.com/excel/newsgroups/data_table/index.2.gif
  2. Javascript/jQuery: create a new subpage and integrate a small picture gallery or a "Slider" with jQuery or Javascript
  3. Webserver: implement another webserver other than Node.js (it doesn't matter which language it's based on -- maybe Apache or HTTP Simple Server?)
We may also add any other feature of our choosing so long as it is relevant to what we learned during the course.

I decided to tackle two tasks: the integration of a table background a slider gallery using jQuery. The documentation of how I implemented both can be found at the bottom of each page. All significant changes made to this website have been tracked via github .
