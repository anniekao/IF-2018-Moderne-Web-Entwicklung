Here I'll explain in detail how this table was created with HTML.

The above table consists of...

  1. <table> element with the following Bootstrap classes:
    • .table-sm: makes the table more compact by cutting the padding in half.
    • .table-hover: the column that the cursor is hovering over is given a grey-highlighted effect.
    • .table-bordered: all sides of the table and cells are given a border.
  2. <thead> defines the headings of the top row. It has been given the class .thead-dark so that the cell backgrounds are black.
  3. <tbody> contains the main content of the table.
  4. Each table row is defined by a <tr> element that is then followed by:
    • <th> element that defines the header of each row. It is given a scope of col to indicate that it is the header for the row
    • <td> elements that contain the data/content of each cell in the row.
  5. Closing tags are required for all elements since none are empty.
