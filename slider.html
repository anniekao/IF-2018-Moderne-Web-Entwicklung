Credit point assignment #2: jQuery Image Slider

Create a image slider using either Javascript or jQuery.

Documentation

Here I'll explain how I used HTML, jQuery, and CSS to create this slider. It should be noted that this slider uses the jQuery plugin slick.

HTML

In the <head> element, the <link> tag references to two external stylesheets needed for slick:

  • slick-theme.min.css
  • slick.min.css

In the <body> element, a <div> with the class .carousel-container is created that contains the <div> with the class .mySlider. Within .mySlider six <div> elements have been created that contain <img> tags that point to the images that will be featured in the slider.

At the bottom of the HTML file, <script> elements are used to point to external script files required for slick to be implemented:

  • jquery.min.js
  • slick.min.js
  • As well as local js file my-javascript.js that will contain the jQuery to initialise the slider.
